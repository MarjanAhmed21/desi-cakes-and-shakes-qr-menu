# Desi Cakes & Shakes - Digital QR Menu

A modern, responsive digital QR menu built for **Desi Cakes & Shakes**, designed to replace traditional printed menus with an interactive mobile-first experience.

## 🍰 Project Overview

This project is a fully responsive digital menu that allows customers to:

- Browse food categories (Mains, Cakes, Milkshakes, Drinks, Desserts, etc.)
- View detailed item cards with images, descriptions, and prices
- Open a modal popup for each item with more information
- Search through the menu instantly
- Navigate smoothly using a sticky category scroll bar
- Place orders directly via WhatsApp integration

The design is optimised for both **mobile (QR scan usage)** and **desktop viewing**.

## 📱 How It Works

1. Customers scan a QR code placed in-store and on windows/leaflets
2. The digital menu loads instantly on their device
3. They browse categories and view items
4. They tap an item to open a modal with details
5. They click “Order on WhatsApp” to place an order instantly
